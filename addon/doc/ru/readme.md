Это дополнение [NVDA](https://nvaccess.org/) добавляет команду для объявления названия, версии, а теперь и архитектуры приложения переднего плана или копирует эту информацию в буфер обмена.

* Сочетание клавиш на английском языке: `NVDA+shift+v`
* Нажмите его один раз, чтобы прослушать произносимую информацию, или получить её в брайлевском виде.
* Нажмите его дважды, чтобы скопировать название, версию и архитектуру в буфер обмена, разделив их переносами строки.
* Нажмите его три раза, чтобы скопировать в буфер обмена только версию и архитектуру.

Эта команда может быть переназначена в категории "Сервис" жестов ввода NVDA.

Чтобы получить доступ к версии Windows, воспользуйтесь командой на рабочем столе или в панели задач.
Она также будет работать из различных других частей Windows.

Чтобы получить доступ к версии NVDA, используйте команду, пока открыто меню NVDA, магазин дополнений или подобный интерфейс NVDA.
