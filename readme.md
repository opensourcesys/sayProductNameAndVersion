This [NVDA](https://nvaccess.org/) add-on adds a shortcut to announce the name and version of the foreground application, or copies this information to the clipboard.

* English shortcut: `NVDA+shift+v`
* Press it once to hear the information spoken, or receive it in braille.
* Press it twice to copy name and version to the clipboard.
* Press it three times to copy only the version to the clipboard.

This shortcut may be remapped in the Tools category of NVDA's Input Gestures.

**Note:** this add-on used to be called "Say Product Name and Version".
While you may still see the word "Product" in the name of some files and in some listings of the add-on name, that is just for technical convenience.
The name has changed to "Say Application Name and Version", or "Say App Name and Version", for short.
This was at the request of some users.
This add-on will likely soon be integrated into NVDA itself, and this change was also made to allow similarity between the built-in NVDA terminology, and the terminology used by people still running the last version of the add-on.

### Tips

* To find out the Windows version with this add-on, you can focus on any Windows component (such as desktop, the start menu, the run dialog, etc.), and press the shortcut key. Note, that this does not give you the release version name, such as "23H3". Instead it gives you something closer to the build version, as you would get from the second line of the "`winver`" command.
* To get the NVDA version, open any NVDA dialog or menu, and press this add-on's shortcut key. For example, press `NVDA+n` to open the NVDA menu, then press `NVDA+shift+v`, then press `escape` to leave the menu.
